# RaspberryPi Shutdown Button

特定のボタンを押すと，Raspberry Pi のシャットダウンを行うためのスクリプト

## How to install

```bash
sudo apt update && \
sudo apt install -y python3-gpiozero git && \
cd /path/to/install/ && \
git clone https://github.com/sik103/rpi_shutdown_button.git && \
cd rpi_shutdown_button && \
bash install.sh
```
